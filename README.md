# 🎤 AI Multilingual Voice Interview Partner

AI-powered **voice-based interview simulator** that helps users practice real-life interviews through natural spoken conversations in multiple languages.

---

## 🚀 Overview

This project is a real-time **AI voice interviewer** that:
- Conducts interviews using **speech (not text)**
- Supports **multiple languages**
- Provides **instant feedback**
- Adapts questions based on user responses

🔗 **Live App (AI Studio):**  
https://ai.studio/apps/7c996753-a9fd-4adc-967e-3edee6c7e6a9

---

## 🎯 Features

- 🎙️ Real-Time Voice Interaction  
- 🌍 Multilingual Support (English, German, Spanish…)  
- 🧠 AI-Powered Interview Questions  
- 🔄 Context-Aware Follow-Ups  
- 📊 Instant Feedback (fluency, clarity, relevance)  
- 🎯 Adaptive Difficulty  
- 🗣️ Natural Human-Like Conversation

## 🧩 How It Works

1. 🎤 User speaks via microphone  
2. 🔤 Speech → Text (STT)  
3. 🤖 AI processes the response  
4. 💬 AI generates next question  
5. 🔊 Text → Speech (TTS)  
6. 🔁 Conversation continues  

---

## 🏗️ Tech Stack

- **Frontend:** React / Next.js  
- **Backend:** Node.js  
- **AI Model:** Gemini / LLM  
- **Speech-to-Text:** Whisper / STT APIs  
- **Text-to-Speech:** TTS APIs  
- **Platform:** AI Studio

## ⚙️ Run Locally

### 📌 Prerequisites
- Node.js installed

---

### 🛠️ Installation

```bash
git clone https://github.com/your-username/voice-interview-ai.git
cd voice-interview-ai
npm install
```

---

### 🔑 Environment Variables

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_api_key
```

---

### ▶️ Start the App

```bash
npm run dev
```

Then open:
```
http://localhost:3000
```

## 🧠 AI Behavior
